__Проект: Вычислитель отличий__ 
===========
    Выполнено в рамках учебного проекта на Hexlet.
    Вычислитель отличий – программа, определяющая разницу между двумя структурами данных форматов JSON, YAML, INI. 

__Установка:__
  $ make install
